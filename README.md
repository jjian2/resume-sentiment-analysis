# 🤖 자기소개서 감정 분석 시스템

## 📌 프로젝트 정보
- 기간 : 2025.04 ~ 2025.06  
- 기관 : 동양미래대학교  
- 기여도 : 100%

## 📖 프로젝트 소개
KoBERT 기반 자연어 처리 모델을 활용하여 자기소개서 문장의 감정(긍정/부정/중립)과 자신감 표현을 자동 분석하는 웹 서비스입니다.  
로그인 시스템을 기반으로 문서 업로드, 분석 결과 저장, 시각화 및 PDF 다운로드 기능을 제공합니다.

## 🧑‍💻 담당 역할
- KoBERT 감정 및 자신감 분류 모델 학습 및 튜닝  
- Flask 기반 AI 서버 구축  
- 문서 업로드 및 분석 파이프라인 구현  
- 결과 시각화 UI 개발  

## 📚 Lesson Learned
- KoBERT fine-tuning 및 실서비스 적용 경험  
- Flask API 설계 및 CORS 문제 해결  
- AI 모델과 웹 서비스 통합 구조 이해
