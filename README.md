# 🤖 자기소개서 감정 분석 시스템

## 📌 프로젝트 정보
- 기간 : 2025.04 ~ 2025.06  
- 기관 : 동양미래대학교  
- 참여 형태 : 개인 프로젝트  
- 기여도 : 100%

## 📖 프로젝트 소개
KoBERT 기반 자연어 처리 모델을 활용하여 자기소개서 문장의 감정(긍정/부정/중립)과 자신감 표현을 자동 분석하는 웹 서비스입니다.  

로그인 시스템을 기반으로 문서 업로드, 분석 결과 저장, 시각화 및 PDF 다운로드 기능을 제공하여  
AI 모델을 실서비스 형태로 구현하는 것을 목표로 개발하였습니다.

## 🧑‍💻 담당 역할
- 서비스 기획 및 전체 시스템 구조 설계  
- KoBERT 감정 및 자신감 분류 모델 학습 및 튜닝  
- Flask 기반 AI 서버 구축  
- 문서 업로드 및 분석 파이프라인 구현  
- 결과 시각화 UI 개발 및 사용자 경험 개선  

## 📚 Lesson Learned
- KoBERT fine-tuning 및 모델 성능 개선 경험  
- Flask 기반 AI API 서버 설계 및 운영 경험  
- CORS 및 인코딩 이슈 해결 실무 경험  
- AI 모델을 웹 서비스로 배포하는 전체 파이프라인 구축 경험
